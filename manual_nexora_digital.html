1. Pengenalan Sistem

Nexora Digital adalah platform toko digital otomatis yang mengintegrasikan Dashboard Admin berbasis Web dengan Bot Telegram. Sistem ini memungkinkan penjualan produk digital — seperti akun premium dan e-book/file digital — secara otomatis 24 jam tanpa harus memproses setiap transaksi secara manual.

Komponen Utama

KomponenFungsi
Dashboard Admin (Web)Mengelola produk, memantau transaksi, verifikasi pembayaran
Bot TelegramMelayani pembeli: browsing katalog, pemesanan, pembayaran
Sistem Pengiriman OtomatisMengirimkan produk digital langsung ke pembeli via Telegram setelah pembayaran terverifikasi

3. Panduan Admin — Setup Awal

Berikut adalah langkah-langkah yang harus dilakukan admin sebelum sistem dapat menerima transaksi dari pembeli.

3.1 Login ke Dashboard

  1. Buka URL dashboard: https://domain-anda/admin/login
  2. Masukkan Email dan Password admin.
  3. Klik tombol Login.

3.2 Konfigurasi Pengaturan Umum (Settings)

Masuk ke menu Settings (ikon ⚙️ di sidebar).

KolomPenjelasan
Nama TokoNama toko yang ditampilkan di bot Telegram
Deskripsi TokoDeskripsi singkat toko
Telegram ID AdminID Telegram akun admin untuk menerima notifikasi pembayaran (wajib diisi)
Pesan Sambutan BotTeks yang dikirim bot saat pembeli pertama kali mengetik /start
Pesan Bantuan BotTeks balasan bot saat pembeli menekan tombol ❓ Bantuan
Info Cara PembayaranTeks petunjuk pembayaran yang dikirim ke pembeli
Kontak AdminTeks berisi cara menghubungi admin, dikirim via tombol ☎️ Hubungi Admin
Cara Mendapatkan Telegram ID Admin:
1. Buka aplikasi Telegram dan cari bot @userinfobot.
2. Ketik /start, bot akan membalas dengan angka ID Telegram Anda.
3. Salin angka tersebut dan isikan di kolom Telegram ID Admin.
Penting: Pastikan akun Telegram admin sudah pernah mengirim /start ke bot toko Anda terlebih dahulu.

3.3 Setup QRIS (QRIS)

Masuk ke menu QRIS (ikon QR Code di sidebar).

  1. Isi kolom Nama Rekening/Akun (nama yang ditampilkan di bawah gambar QRIS ke pembeli).
  2. Upload gambar QRIS Anda (format: JPG/PNG, maksimal 2 MB).
  3. Klik Simpan.
Gambar QRIS ini akan otomatis dikirimkan bot ke pembeli saat pembeli diminta melakukan pembayaran.

4. Manajemen Katalog

4.1 Kategori (Kategori)

Kategori berfungsi sebagai pengelompokan produk di toko Telegram Anda.

Cara Membuat Kategori:

  1. Masuk ke menu Kategori.
  2. Klik tombol Tambah Kategori.
  3. Isi Nama Kategori dan Ikon (emoji, misal: 🎮).
  4. Aktifkan status Aktif agar kategori muncul di bot.
  5. Klik Simpan.
Tips: Buat kategori yang spesifik dan jelas, misalnya: Akun Netflix, Akun Spotify, E-book Bisnis.

4.2 Produk (Produk)

Produk adalah item yang dijual di dalam sebuah kategori.

Cara Membuat Produk:

  1. Masuk ke menu Produk.
  2. Klik tombol Tambah Produk.
  3. Isi data produk:
KolomPenjelasan
Nama ProdukNama yang tampil di bot Telegram
KategoriPilih kategori yang sudah dibuat
Tipe Produkaccount (akun digital) atau file/ebook
DeskripsiPenjelasan singkat produk
StatusAktif/Nonaktif

Perbedaan Tipe Produk:
- account → Produk berupa kredensial akun (username + password). Stok dihitung otomatis dari data akun yang diupload.
- file / ebook → Produk berupa file digital atau teks. Stok diisi manual.

4.3 Varian Produk

Satu produk bisa memiliki beberapa varian (misal: Netflix 1 Bulan, Netflix 3 Bulan, Netflix 1 Tahun).

Cara Membuat Varian:

  1. Buka halaman detail produk → klik Kelola Varian.
  2. Klik Tambah Varian.
  3. Isi data varian:
KolomPenjelasan
Nama VarianContoh: "1 Bulan", "3 Bulan"
HargaHarga jual varian
Harga Asli(Opsional) Harga coretan untuk menampilkan diskon
StokHanya untuk produk tipe file; tipe account dihitung otomatis
StatusAktif/Nonaktif

4.4 Stok Akun (Khusus Produk Tipe account)

Untuk produk bertipe akun, stok diisi dengan cara mengupload daftar kredensial.

Cara Upload Kredensial Akun:

  1. Buka halaman detail varian → klik Kelola Stok Akun.
  2. Di kolom teks yang tersedia, masukkan daftar akun dengan format:
email_atau_username:password
   email_atau_username:password
Contoh: 
user1@gmail.com:pass123
   user2@gmail.com:pass456
   akun3@email.com:mypassword
  1. Klik Tambah Akun.
  2. Sistem akan otomatis menghitung stok berdasarkan jumlah akun yang belum terjual.
Info Stok:
- Akun berstatus Tersedia: belum terjual, siap dikirim ke pembeli.
- Akun berstatus Terjual: sudah dikirim ke pembeli, tidak bisa digunakan lagi.
- Admin bisa menghapus akun yang salah input.

4.5 File Digital (Digital Files)

Untuk produk bertipe file/ebook, admin perlu mengupload file atau mengatur konten pengiriman.

Cara Setup Digital File:

  1. Masuk ke menu Digital Files.
  2. Klik Tambah File Digital atau asosiasikan file ke varian produk.
  3. Pilih Tipe Pengiriman:
TipeCara Kerja
FileUpload file (PDF, ZIP, dll). Bot akan mengirim file langsung ke pembeli.
Teks OtomatisTulis teks berisi link download, kode lisensi, dll. Bot kirim teks ke pembeli.
ManualAdmin harus mengirim produk secara manual setelah pembayaran terverifikasi.

5. Manajemen Transaksi

5.1 Order (Order)

Menu Order menampilkan seluruh daftar pesanan yang masuk dari pembeli.

Kolom Informasi Order:

  • Invoice — Nomor unik pesanan
  • Pembeli — Nama user Telegram
  • Produk & Varian — Produk yang dipesan
  • Total — Nominal pembayaran
  • Status — Status pesanan saat ini
  • Tanggal — Waktu pesanan dibuat
Aksi yang Tersedia:
  • Klik tombol Detail untuk melihat informasi lengkap pesanan.
  • Batalkan pesanan jika diperlukan (admin manual).

5.2 Payment (Verifikasi Pembayaran)

Menu Payment menampilkan daftar semua pembayaran yang masuk beserta bukti transfer dari pembeli.

Verifikasi via Web Admin

  1. Klik Detail pada pembayaran yang berstatus Menunggu Verifikasi.
  2. Periksa foto bukti pembayaran dari pembeli.
  3. Klik tombol:
- ✅ Verifikasi jika pembayaran valid → sistem otomatis mengirim produk ke pembeli. - ❌ Tolak jika pembayaran tidak valid → masukkan alasan penolakan → sistem otomatis notifikasi ke pembeli.

Verifikasi via Telegram Admin (Cara Cepat)

Saat pembeli mengunggah bukti bayar, bot akan langsung mengirimkan notifikasi ke Telegram Admin berupa:
  • Foto bukti pembayaran
  • Detail pesanan (invoice, produk, nominal, pembeli)
  • Tombol [ ✅ Verifikasi ] dan [ ❌ Tolak ]
Admin cukup menekan tombol langsung di Telegram tanpa harus membuka web admin.
Proteksi Klik Ganda: Jika tombol ditekan lebih dari sekali setelah pesanan berhasil diproses, sistem akan menampilkan notifikasi bahwa pesanan sudah diproses dan tidak akan memproses ulang.

5.3 QRIS

Menu QRIS digunakan untuk mengelola gambar kode QR pembayaran yang dikirimkan ke pembeli.

6. Manajemen Telegram

6.1 Telegram Users

Menu Telegram Users menampilkan seluruh pengguna yang pernah berinteraksi dengan bot.

Fitur yang Tersedia:

  • Cari pengguna berdasarkan nama atau username.
  • Filter berdasarkan status (aktif / diblokir).
  • Lihat Detail User: informasi profil, riwayat pesanan, status sesi bot.
  • Blokir / Buka Blokir User: pengguna yang diblokir tidak dapat menggunakan bot.

6.2 Broadcast

Fitur Broadcast memungkinkan admin mengirim pesan massal ke semua pengguna bot.

Cara Mengirim Broadcast:

  1. Masuk ke menu Broadcast.
  2. Tulis pesan yang ingin dikirim (mendukung teks panjang hingga 4.096 karakter).
  3. Pilih target penerima:
- Semua User — Dikirim ke seluruh pengguna bot yang tidak diblokir. - User Aktif — Hanya dikirim ke pengguna yang sudah pernah melakukan pemesanan.
  1. Klik Kirim Broadcast.
Catatan: Broadcast dikirim via sistem antrean (Queue) agar tidak membebani server. Jumlah penerima yang berhasil dinotifikasi akan ditampilkan setelah proses selesai.

7. Pengaturan Sistem

7.1 Settings (Pengaturan)

(Sudah dijelaskan di bagian 3.2 — Setup Awal)

7.2 Activity Log

Menu Activity Log mencatat seluruh tindakan yang dilakukan oleh admin di dalam sistem, termasuk:

  • Verifikasi dan penolakan pembayaran
  • Penambahan/perubahan produk
  • Upload akun/stok
  • Pengiriman broadcast
  • Perubahan pengaturan
  • Blokir/unblokir user Telegram
Setiap log mencatat: Aksi, Deskripsi, IP Address, dan Waktu.

7.3 Admin (Manajemen Akun Admin)

Menu Admin digunakan untuk mengelola akun admin yang memiliki akses ke dashboard.

Fitur:

  • Tambah Admin Baru: Buat akun admin tambahan dengan email dan password baru.
  • Edit Admin: Ubah nama, email, atau password admin.
  • Hapus Admin: Hapus akun admin yang tidak diperlukan lagi.
Keamanan: Setiap akun admin memiliki akses penuh ke dashboard. Pastikan hanya memberikan akses kepada pihak yang dipercaya.

8. Panduan Pembeli via Bot Telegram

Bagian ini menjelaskan cara penggunaan bot dari sudut pandang pembeli.

8.1 Memulai Bot

  1. Cari bot di Telegram (contoh: @NexoraDigitalStoreBot).
  2. Klik START atau ketik /start.
  3. Bot akan menampilkan Menu Utama dengan pilihan:
- 🛍️ Produk - 📦 Pesanan Saya - 💳 Cara Pembayaran - ❓ Bantuan - ☎️ Hubungi Admin

8.2 Alur Pembelian

Langkah 1: Pilih Produk

  1. Tekan tombol 🛍️ Produk.
  2. Pilih Kategori yang tersedia.
  3. Pilih Produk yang diinginkan.
  4. Pilih Varian (misal: 1 Bulan, 3 Bulan, 1 Tahun).
Jika varian sedang habis stok, sistem otomatis menampilkan pesan "Mohon maaf, varian produk ini sedang habis."

Langkah 2: Isi Catatan Pesanan

Bot akan meminta informasi tambahan (misal: username Instagram, email, nomor HP). Isi sesuai kebutuhan produk yang dibeli.

Langkah 3: Konfirmasi Order

Bot menampilkan ringkasan pesanan:

  • Nama produk & varian
  • Harga
  • Catatan pesanan
Tekan ✅ Konfirmasi Order untuk melanjutkan, atau ❌ Batalkan Pesanan untuk membatalkan.

Langkah 4: Pembayaran

Bot akan mengirimkan:

  • Gambar kode QRIS untuk pembayaran
  • Nominal yang harus ditransfer
  • Instruksi pembayaran
Lakukan transfer sesuai nominal yang tertera.

Langkah 5: Upload Bukti Pembayaran

Setelah transfer, kirim foto/gambar bukti pembayaran langsung ke chat bot.

Bot akan mengkonfirmasi bahwa bukti sudah diterima dan sedang diverifikasi.

Langkah 6: Terima Produk

Setelah admin memverifikasi pembayaran, bot akan otomatis mengirimkan produk ke chat pembeli:

  • Akun digital: Username dan password dikirim langsung.
  • File digital: File dikirim langsung ke chat.
  • Teks/Lisensi: Kode atau link dikirim dalam bentuk pesan teks.

8.3 Cek Riwayat Pesanan

Pembeli dapat melihat 5 pesanan terakhir dengan menekan tombol 📦 Pesanan Saya di menu utama.

8.4 Pembayaran Ditolak?

Jika admin menolak bukti pembayaran:

  1. Pembeli akan menerima notifikasi penolakan dari bot.
  2. Pesanan kembali ke status Menunggu Pembayaran.
  3. Pembeli perlu mengirim ulang bukti pembayaran yang valid.
